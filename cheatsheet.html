Matt's Cheatsheet for HTML Tables

HTML Table Tags

Tag Name Purpose
<table> Table The wrapper for all table elements
<thead> Table Head Defines headings of table columns
<tbody> Table Body A semantic element that will contain everything but the header and footer of a table
<tr> Table Row Used to add rows to the table
<td> Table Data Nested inside a row, it adds a cell with data to the row
<tfoot> Table Foot Uses rows to give footer content or a summary of the table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a cell will span
rowspan Row Span Defines how many rows a cell will span
